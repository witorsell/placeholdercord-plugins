# UwUify

Lets you UwUify text, as a filter on all your outgoing messages or through the `/uwuify` command. Fork of [UwUify](https://github.com/meqativ/dumsane/tree/master/plugins/UwUify) by meqativ.

## Install

Paste this URL into the Plugins page of Kettu/Vendetta:

> https://witorsell.github.io/uwuify-live/

## What is fixed

Settings now apply instantly. The uwuifier is rebuilt from your current settings on every use, so toggling faces, actions, stutters, words, or exclamations takes effect immediately. No more "Reload uwuifier" button and no Discord restart needed.
